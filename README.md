# XQuest Leaderboard API

The back end for the XQuest Leaderboard Application

## Getting Started

 Clone this repository into a directory of your choosing

### Prerequisites



Ruby/ Ruby on Rails 
Postgresql

### Installing

In the project directory run 

1. bundle install
2. rails s -p 3001
